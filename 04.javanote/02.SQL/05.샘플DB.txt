[ 샘플 DB 설치하기 ]

1. North Wind 라는 샘플 DB 설치하기

2. NorthWind DB소개
- MS Access 프로그램 기본 DB (2010년 버전)
- 설치 기본형은 MySQL용
- Northwind 는 고객, 주문, 재고, 공급업체, 배송 등의 소규모 비즈니스 DB

3. 구글 검색으로 NorthWind 쿼리문 찾기
> "mysql northwind sample database" 로 검색
> 결과 : 첫번째 결과링크 > dalers/mywind.. 깃허브 사이트

[ northwind 킷허브 주소 ]
> https://github.com/dalers/mywind

> northwind.sql : 테이블 생성 쿼리문
> northwind-data.sql : 데이터 삽입 쿼리문

4. 나의 MySQL에서 DB를 생성후 거기에 쿼리를 실행한다.
- 쿼리문을 메인 SQL에서 실행하면 "Northwind" DB 가 만들어지고 거기에 각 테이브링 생성된다.
